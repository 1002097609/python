## 维度复用矩阵

> 哪些维度适合作为"骨架"（可复用），哪些适合作为"叶子"（可替换）？

### 适合作为骨架的维度（跨素材复用）

```
✅ 内容策略（第二章）— 诉求策略、情绪策略、说服逻辑
✅ 叙事结构（第四章）— 宏观结构、开头/主体/结尾模式
✅ 表达风格中的句式特征（第五章）— 句式模式、人称使用
✅ 平台适配中的推荐机制偏好（第六章）— 算法逻辑
✅ 转化路径（第七章）— 用户决策链路
✅ 素材关系中的裂变模式（第十一章）— 裂变方法本身
```

### 适合作为叶子的维度（裂变时替换）

```
🔄 目标人群（第一章）— 年龄/性别/地域/场景/圈层
🔄 内容主题（第三章）— 品类/产品/卖点/话题
🔄 表达风格中的人设和语言（第五章）— 风格/语气/视觉/音频
🔄 平台适配中的平台特性（第六章）— 平台/格式/话题标签
🔄 投放运营中的定向和竞价（第七章）— 人群/地域/时间/出价
🔄 时间维度（第九章）— 季节/节日/大促/时段
🔄 合规与品牌（第十章）— 法规/调性/品牌元素
🔄 素材关系（第十一章）— 系列/版本/母体
🔄 创作者（第十二章）— 创作者类型/量级/风格
🔄 效果数据（第八章）— 随投放结果动态变化
```

## 总结：维度与五层模型的映射

| 五层模型 | 对应维度 | 角色 |
|----------|---------|------|
| **L1 主题层** | 内容主题（第三章）| 可替换（换品类/产品/卖点） |
| **L2 策略层** | 内容策略（第二章）| **可复用**（骨架核心） |
| **L3 结构层** | 叙事结构（第四章）| **可复用**（骨架核心） |
| **L4 元素层** | 表达风格中的可插拔模块（第五章）| **可复用**（可插拔） |
| **L5 表达层** | 目标人群 + 具体表达（第一章+第五章细节）| 可替换（换人群/换内容） |

**关键洞察**：
- 现有五层模型（L1-L5）是**内容视角**的拆解，覆盖了约 **40%** 的可拆分维度
- 还有大量维度在现有模型之外：**目标人群、平台适配、投放运营、时间、合规、创作者、效果数据、素材关系**
- 这些"内容之外"的维度同样可以成为骨架的一部分——比如"平台适配策略"本身就是一个可复用的骨架维度
- 未来系统迭代时，可以从这些维度中选取高价值的部分纳入骨架体系
