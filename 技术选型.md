---

## 决策依据

| 考量因素 | 结论 |
|---------|------|
| **技术背景** | 主 Python → FastAPI 开发效率高，异步性能优于 Flask |
| **数据库需求** | 明确要求 MySQL → 关系型数据 + JSON 字段，完美匹配 |
| **前端需求** | React/Vue 均可 → 推荐 Vue 3，prototype.html 本身就是 Vue 风格模板，迁移成本低 |
| **AI 接入** | Python 生态 → P3 阶段接 LangChain/OpenAI SDK 无缝衔接 |
| **工程质量** | FastAPI 自带自动 API 文档（Swagger）、类型校验、异步支持，适合长期维护 |

## 技术栈详设

### 后端

| 选型 | 版本 | 说明 |
|------|------|------|
| **Python** | 3.11+ | 主语言 |
| **FastAPI** | 0.110+ | 异步 Web 框架，自动 Swagger 文档 |
| **SQLAlchemy** | 2.x | ORM，支持 MySQL JSON 字段 |
| **PyMySQL** / **aiomysql** | 最新 | MySQL 驱动（同步/异步） |
| **Pydantic** | 2.x | FastAPI 内置，请求/响应数据校验 |
| **uvicorn** | 最新 | ASGI 服务器，运行 FastAPI |
| **Alembic** | 最新 | 数据库迁移工具（后续迭代用） |

### 前端

| 选型 | 说明 |
|------|------|
| **Vue 3** | 组合式 API (`<script setup>`)，开发体验好 |
| **Vite** | 构建工具，开箱即用，热更新极快 |
| **Axios** | HTTP 客户端，调后端 REST API |
| **Vue Router** | 前端路由（多页面切换） |
| **Pinia** | 状态管理（素材/骨架等全局状态） |
| **Element Plus** | UI 组件库（表格/表单/弹窗等），节省样式开发时间 |

### 数据库

| 选型 | 说明 |
|------|------|
| **MySQL 8.0** | JSON 字段支持 L3/L4 结构化数据存储 |
| **utf8mb4** | 字符集，支持 emoji |

### API 风格

```
RESTful API，JSON 请求/响应
FastAPI 自动生成 Swagger 文档：http://localhost:8000/docs
```

## 环境准备

### 需要提前安装

```
✅ Python 3.11+
✅ MySQL 8.0
✅ Node.js 18+
✅ npm / pnpm
```

### MySQL 初始化

```sql
CREATE DATABASE material_system CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER 'material_user'@'localhost' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON material_system.* TO 'material_user'@'localhost';
FLUSH PRIVILEGES;
```

### 后端启动

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --host 0.0.0.0 --port 8000
# Swagger: http://localhost:8000/docs
```

### 前端启动

```bash
cd frontend
npm install
npm run dev
# 默认: http://localhost:5173
```
